# Algoritmos Genéticos - Resolución de Sudokus
El objetivo es construir un algoritmo genético capaz de resolver Sudokus de NxN, es decir, con las mismas filas que 
columnas (cuadrados).

Un algoritmo genético es una técnica heurística de Algoritmo Evolutivo [1], los cuales se centran en la búsqueda de 
soluciones a través de una representación evolutiva del problema. Esta heurística se inspira en el proceso evolutivo 
de los seres vivos.

En la documentación se ha incluido un manual de usuario que explica cómo ejecutar el programa, y la carpeta test contiene 
distintos ejemplos de entrada para el mismo, es decir, sudokus de prueba. El archivo zip contiene las librerías que utiliza 
el programa.

En concreto se hace uso de GAlib que es una librería de objetos implementados en el lenguaje C++ que tiene como objetivo diseñar y 
construir algoritmos genéticos. La librería incluye un conjuntode herramientas para poder utilizar algoritmos genéticos y resolver 
diferentes problemas de optimización utilizando diferentes representaciones y operadores genéticos.
